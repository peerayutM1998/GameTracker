# 🎮 GameTracker

GameTracker is a cross-platform application built with **.NET MAUI**. This application is designed with a modern, responsive user interface supporting both Light and Dark themes, providing a seamless experience across multiple devices.

## 🚀 Features
* **Cross-Platform Support:** Runs natively on Android, iOS, Windows, and macOS from a single codebase.
* **Modern UI/UX:** Custom theming and styling utilizing XAML resources (`Styles.xaml`), including full Light/Dark mode support.
* **Clean Navigation:** Customized Shell navigation with options for hidden Action Bars for a truly immersive app experience.

## 🛠️ Tech Stack
* **Framework:** [.NET MAUI](https://dotnet.microsoft.com/en-us/apps/maui) (Multi-platform App UI)
* **Language:** C# / XAML
* **Version Control:** Git / GitHub

## 📋 Prerequisites
To run this project locally, ensure you have the following installed:
* [Visual Studio 2022](https://visualstudio.microsoft.com/) (with the **.NET Multi-platform App UI development** workload installed) OR [Visual Studio Code](https://code.visualstudio.com/) with the .NET MAUI extension.
* .NET SDK (7.0 or 8.0)
